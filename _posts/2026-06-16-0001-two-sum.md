## 题目描述

给定一个整数数组 `nums` 和一个目标值 `target`，请你在数组中找出和为目标值的那两个整数，并返回它们的下标。假设每种输入只对应一个答案，且同一元素不能用两遍。

**示例：** `nums = [2,7,11,15], target = 9` → `[0,1]`（因为 `nums[0]+nums[1]=9`）。

## 思路分析

**核心思想：** 用哈希表记录"已经见过的数字 → 它的下标"，对当前数字 `num`，只需查 `target - num` 是否已出现。

**详细步骤：**

1. 建空哈希表 `{数字: 下标}`。
2. 遍历每个 `num`（下标 `i`）：查 `target - num`——在表里就返回两个下标；否则把 `{num: i}` 存进表。
3. 把"找配对"从 O(n) 降到 O(1)，总体一次遍历即可。

func twoSum(nums []int, target int) []int {
    m := make(map[int]int) // 数字 -> 下标
    for i, num := range nums {
        if j, ok := m[target-num]; ok {
            return []int{j, i}
        }
        m[num] = i
    }
    return nil
}
```

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        seen = {}  # 数字 -> 下标
        for i, num in enumerate(nums):
            if target - num in seen:
                return [seen[target - num], i]
            seen[num] = i
        return []
```
