## 题目描述

机器人位于 `m × n` 网格左上角，每次只能向右或向下移动，求到达右下角的不同路径数。

**示例：** `m = 3, n = 7` → `28`。

## 思路分析

**核心思想：** 到达 `(i, j)` 的路径数 = 上方 + 左方。用一维滚动数组：`dp[j] += dp[j-1]`。

**详细步骤：**

1. 首行全为 1（只能一路向右）。
2. 逐行更新：`dp[j] = dp[j]（上） + dp[j-1]（左）`。

```go
func uniquePaths(m int, n int) int {
    dp := make([]int, n)
    for j := range dp {
        dp[j] = 1
    }
    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            dp[j] += dp[j-1] // 上 + 左
        }
    }
    return dp[n-1]
}
```

<div class="tab-content" data-lang="python" markdown="1">

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1] * n
        for _ in range(1, m):
            for j in range(1, n):
                dp[j] += dp[j - 1]
        return dp[-1]
```
