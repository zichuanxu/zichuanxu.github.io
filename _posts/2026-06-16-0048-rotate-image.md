## 题目描述

给定 `n × n` 矩阵表示图像，将其原地顺时针旋转 90°。

**示例：** `[[1,2,3],[4,5,6],[7,8,9]]` → `[[7,4,1],[8,5,2],[9,6,3]]`。

## 思路分析

**核心思想：** 顺时针旋转 90° = 先沿主对角线**转置**，再把每一行**水平反转**。

**详细步骤：**

1. 转置：交换 `matrix[i][j]` 与 `matrix[j][i]`（只遍历上三角）。
2. 反转：把每一行首尾对调。

```go
func rotate(matrix [][]int) {
    n := len(matrix)
    for i := 0; i < n; i++ { // 转置
        for j := i + 1; j < n; j++ {
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        }
    }
    for i := 0; i < n; i++ { // 每行反转
        for l, r := 0, n-1; l < r; l, r = l+1, r-1 {
            matrix[i][l], matrix[i][r] = matrix[i][r], matrix[i][l]
        }
    }
}
```

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        for i in range(n):                      # 转置
            for j in range(i + 1, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        for row in matrix:                      # 每行反转
            row.reverse()
```
