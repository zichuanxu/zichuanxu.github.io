## 题目描述

给定不含重复元素的数组 `nums`，返回其所有可能的子集（幂集）。

**示例：** `nums = [1,2,3]` → `[[],[1],[2],[3],[1,2],[1,3],[2,3],[1,2,3]]`。

## 思路分析

**核心思想：** 回溯。每到一个节点就把当前路径记为一个子集；从 `start` 起依次选择后续元素，保证不重复、不回头。

**详细步骤：**

1. 进入递归先收集当前路径（含空集）。
2. 从 `start` 遍历：选入元素、递归 `i+1`、再撤销。

```go
func subsets(nums []int) [][]int {
    res := [][]int{}
    path := []int{}
    var backtrack func(start int)
    backtrack = func(start int) {
        res = append(res, append([]int{}, path...))
        for i := start; i < len(nums); i++ {
            path = append(path, nums[i])
            backtrack(i + 1)
            path = path[:len(path)-1]
        }
    }
    backtrack(0)
    return res
}
```

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res, path = [], []
        def backtrack(start):
            res.append(path[:])
            for i in range(start, len(nums)):
                path.append(nums[i])
                backtrack(i + 1)
                path.pop()
        backtrack(0)
        return res
```
