## 题目描述

给定两个单词 `word1`、`word2`，返回把 `word1` 转换成 `word2` 所需的最少操作数（可插入、删除、替换一个字符）。

**示例：** `word1 = "horse", word2 = "ros"` → `3`。

## 思路分析

**核心思想：** 二维 DP。`dp[i][j]` 为 `word1[:i]` 变成 `word2[:j]` 的最少操作。字符相等则继承左上；否则由「替换/删除/插入」三者取最小 +1。

**详细步骤：**

1. 边界：`dp[i][0] = i`（全删）、`dp[0][j] = j`（全插）。
2. 相等：`dp[i][j] = dp[i-1][j-1]`。
3. 不等：`dp[i][j] = 1 + min(dp[i-1][j-1] 替换, dp[i-1][j] 删除, dp[i][j-1] 插入)`。

```go
func minDistance(word1 string, word2 string) int {
    m, n := len(word1), len(word2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
        dp[i][0] = i
    }
    for j := 0; j <= n; j++ {
        dp[0][j] = j
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if word1[i-1] == word2[j-1] {
                dp[i][j] = dp[i-1][j-1]
            } else {
                dp[i][j] = 1 + min(dp[i-1][j-1], min(dp[i-1][j], dp[i][j-1]))
            }
        }
    }
    return dp[m][n]
}
```

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(m + 1):
            dp[i][0] = i
        for j in range(n + 1):
            dp[0][j] = j
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    dp[i][j] = 1 + min(dp[i - 1][j - 1], dp[i - 1][j], dp[i][j - 1])
        return dp[m][n]
```
