## 题目描述

给定数组 `nums`（保证能到末尾），返回到达最后一个下标的**最少跳跃次数**。

**示例：** `nums = [2,3,1,1,4]` → `2`（`0 → 1 → 4`）。

## 思路分析

**核心思想：** 贪心「区间扩张」。维护当前一跳能覆盖的右边界 `curEnd` 与遍历过程中的最远可达 `farthest`；走到 `curEnd` 时必须再跳一步，并把边界更新为 `farthest`。

**详细步骤：**

1. 遍历到 `len-1` 之前（最后一格无需再跳）。
2. 不断更新 `farthest = max(farthest, i + nums[i])`。
3. 当 `i == curEnd` 时跳数 +1，`curEnd = farthest`。

```go
func jump(nums []int) int {
    jumps, curEnd, farthest := 0, 0, 0
    for i := 0; i < len(nums)-1; i++ {
        if i+nums[i] > farthest {
            farthest = i + nums[i]
        }
        if i == curEnd { // 必须再跳一步
            jumps++
            curEnd = farthest
        }
    }
    return jumps
}
```

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        jumps = cur_end = farthest = 0
        for i in range(len(nums) - 1):
            farthest = max(farthest, i + nums[i])
            if i == cur_end:
                jumps += 1
                cur_end = farthest
        return jumps
```
