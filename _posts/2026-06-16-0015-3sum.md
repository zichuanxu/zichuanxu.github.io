## 题目描述

给定数组 `nums`，找出所有和为 `0` 且**不重复**的三元组。

**示例：** `nums = [-1,0,1,2,-1,-4]` → `[[-1,-1,2],[-1,0,1]]`。

## 思路分析

**核心思想：** 先排序，再固定一个数 `nums[i]`，在其右侧用双指针找两数之和为 `-nums[i]`。

**详细步骤：**

1. 排序数组。
2. 固定 `i`（若 `nums[i] > 0` 可提前结束；与前一个相同则跳过去重）。
3. `l = i+1, r = n-1`：根据三数之和与 0 的大小移动指针；命中时记录并跳过重复元素。

```go
func threeSum(nums []int) [][]int {
    sort.Ints(nums)
    n := len(nums)
    res := [][]int{}
    for i := 0; i < n-2; i++ {
        if nums[i] > 0 {
            break
        }
        if i > 0 && nums[i] == nums[i-1] {
            continue // 跳过重复的固定数
        }
        l, r := i+1, n-1
        for l < r {
            sum := nums[i] + nums[l] + nums[r]
            switch {
            case sum < 0:
                l++
            case sum > 0:
                r--
            default:
                res = append(res, []int{nums[i], nums[l], nums[r]})
                for l < r && nums[l] == nums[l+1] {
                    l++
                }
                for l < r && nums[r] == nums[r-1] {
                    r--
                }
                l++
                r--
            }
        }
    }
    return res
}
```

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        n, res = len(nums), []
        for i in range(n - 2):
            if nums[i] > 0:
                break
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            l, r = i + 1, n - 1
            while l < r:
                s = nums[i] + nums[l] + nums[r]
                if s < 0:
                    l += 1
                elif s > 0:
                    r -= 1
                else:
                    res.append([nums[i], nums[l], nums[r]])
                    while l < r and nums[l] == nums[l + 1]:
                        l += 1
                    while l < r and nums[r] == nums[r - 1]:
                        r -= 1
                    l += 1
                    r -= 1
        return res
```
