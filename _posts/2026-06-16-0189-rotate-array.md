## 题目描述

给定数组 `nums`，将元素向右轮转 `k` 个位置（`k` 非负）。要求原地、O(1) 额外空间。

**示例：** `nums = [1,2,3,4,5,6,7], k = 3` → `[5,6,7,1,2,3,4]`。

## 思路分析

**核心思想：** 三次反转。整体反转后，前 `k` 个与后 `n-k` 个分别反转即可复位。

**详细步骤：**

1. `k %= n`（避免无效轮转）。
2. 反转整个数组。
3. 分别反转 `[0, k-1]` 与 `[k, n-1]`。

```go
func rotate(nums []int, k int) {
    n := len(nums)
    k %= n
    reverse(nums, 0, n-1)
    reverse(nums, 0, k-1)
    reverse(nums, k, n-1)
}

func reverse(nums []int, i, j int) {
    for i < j {
        nums[i], nums[j] = nums[j], nums[i]
        i++
        j--
    }
}
```

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        n = len(nums)
        k %= n
        nums.reverse()
        nums[:k] = reversed(nums[:k])
        nums[k:] = reversed(nums[k:])
```
