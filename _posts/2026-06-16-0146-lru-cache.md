## 题目描述

设计一个 LRU（最近最少使用）缓存，支持 `get(key)` 与 `put(key, value)` 操作。容量满时淘汰最久未使用的项。要求 `get`/`put` 均为 O(1)。

**示例：** 容量为 2，依次 `put(1,1)`、`put(2,2)`、`get(1)`→1、`put(3,3)`（淘汰 key=2）、`get(2)`→-1。

## 思路分析

**核心思想：** 哈希表 + 双向链表。哈希表将 key 映射到链表节点，实现 O(1) 定位；双向链表维护使用顺序（头部 = 最近使用，尾部 = 最久未使用）。

**详细步骤：**

1. 使用 `head`/`tail` 哨兵节点简化边界处理，避免空链表判断。
2. `get` 命中时，将节点移到链表头部（`remove` + `addFront`），返回值；未命中返回 -1。
3. `put` 命中时更新值并移到头部；新增时先插入头部，若超出容量则删除尾部节点并从哈希表中移除对应 key。

```go
type LRUCache struct {
    cap        int
    m          map[int]*node
    head, tail *node
}
type node struct {
    key, val   int
    prev, next *node
}

func Constructor(capacity int) LRUCache {
    h, t := &node{}, &node{}
    h.next, t.prev = t, h
    return LRUCache{cap: capacity, m: map[int]*node{}, head: h, tail: t}
}
func (c *LRUCache) remove(n *node) { n.prev.next = n.next; n.next.prev = n.prev }
func (c *LRUCache) addFront(n *node) {
    n.prev, n.next = c.head, c.head.next
    c.head.next.prev, c.head.next = n, n
}
func (c *LRUCache) Get(key int) int {
    if n, ok := c.m[key]; ok {
        c.remove(n)
        c.addFront(n)
        return n.val
    }
    return -1
}
func (c *LRUCache) Put(key, value int) {
    if n, ok := c.m[key]; ok {
        n.val = value
        c.remove(n)
        c.addFront(n)
        return
    }
    n := &node{key: key, val: value}
    c.m[key] = n
    c.addFront(n)
    if len(c.m) > c.cap {
        last := c.tail.prev
        c.remove(last)
        delete(c.m, last.key)
    }
}
```

```python
class Node:
    def __init__(self, key=0, val=0):
        self.key, self.val = key, val
        self.prev = self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        self.cap = capacity
        self.m = {}
        self.head, self.tail = Node(), Node()
        self.head.next, self.tail.prev = self.tail, self.head

    def _remove(self, n):
        n.prev.next, n.next.prev = n.next, n.prev

    def _add_front(self, n):
        n.prev, n.next = self.head, self.head.next
        self.head.next.prev = self.head.next = n

    def get(self, key: int) -> int:
        if key in self.m:
            n = self.m[key]
            self._remove(n)
            self._add_front(n)
            return n.val
        return -1

    def put(self, key: int, value: int) -> None:
        if key in self.m:
            n = self.m[key]
            n.val = value
            self._remove(n)
            self._add_front(n)
            return
        n = Node(key, value)
        self.m[key] = n
        self._add_front(n)
        if len(self.m) > self.cap:
            last = self.tail.prev
            self._remove(last)
            del self.m[last.key]
```

## 复杂度分析

| | 时间复杂度 | 空间复杂度 |
|---|---|---|
| Go / Python | O(1) per get/put | O(capacity) |

**解释：** 哈希表 O(1) 定位节点，双向链表 O(1) 移动/删除节点；总空间与缓存容量成正比。
