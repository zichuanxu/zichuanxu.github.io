## 题目描述

给定每间房屋金额数组 `nums`，不能偷相邻两间，求一夜能偷到的最大金额。

**示例：** `nums = [2,7,9,3,1]` → `12`（偷 2 + 9 + 1）。

## 思路分析

**核心思想：** 到第 `i` 间的最大收益 = `max(不偷 i = dp[i-1], 偷 i = dp[i-2] + nums[i])`。用两个滚动变量。

**详细步骤：**

1. `prev = dp[i-2]`，`cur = dp[i-1]`，初始均为 0。
2. 每间房：`next = max(cur, prev + nums[i])`，滚动更新。

```go
func rob(nums []int) int {
    prev, cur := 0, 0 // prev = dp[i-2], cur = dp[i-1]
    for _, n := range nums {
        next := cur
        if prev+n > next {
            next = prev + n
        }
        prev, cur = cur, next
    }
    return cur
}
```

```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        prev, cur = 0, 0
        for n in nums:
            prev, cur = cur, max(cur, prev + n)
        return cur
```
