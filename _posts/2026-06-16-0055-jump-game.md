## 题目描述

给定非负整数数组 `nums`，`nums[i]` 是从位置 `i` 能跳的最大步数。判断能否到达最后一个下标。

**示例：** `nums = [2,3,1,1,4]` → `true`；`nums = [3,2,1,0,4]` → `false`。

## 思路分析

**核心思想：** 贪心维护「当前能到达的最远下标」`reach`。遍历时若某下标已超出 `reach` 说明断档；否则用 `i + nums[i]` 扩展 `reach`。

**详细步骤：**

1. `reach = 0`。
2. 遍历 `i`：若 `i > reach` 返回 `false`。
3. 更新 `reach = max(reach, i + nums[i])`；遍历完即可达。

```go
func canJump(nums []int) bool {
    reach := 0
    for i, n := range nums {
        if i > reach {
            return false // 到不了 i
        }
        if i+n > reach {
            reach = i + n
        }
    }
    return true
}
```

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        reach = 0
        for i, n in enumerate(nums):
            if i > reach:
                return False
            reach = max(reach, i + n)
        return True
```
