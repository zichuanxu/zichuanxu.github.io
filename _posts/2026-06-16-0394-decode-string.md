## 题目描述

给定编码字符串，规则为 `k[encoded]` 表示方括号内字符串重复 `k` 次。返回解码后的字符串。

**示例：** `s = "3[a]2[bc]"` → `"aaabcbc"`；`s = "2[abc]3[cd]ef"` → `"abcabccdcdcdef"`。

## 思路分析

**核心思想：** 双栈。遇 `[` 时把当前累计字符串和重复次数分别压栈、清空；遇 `]` 时弹出取重复并拼回。

**详细步骤：**

1. 数字累计成 `num`；字母直接追加到 `cur`。
2. 遇 `[`：压入 `num` 与 `cur`，重置二者。
3. 遇 `]`：弹出次数 `k` 与之前的串 `prev`，令 `cur = prev + cur*k`。

```go
func decodeString(s string) string {
    countStack := []int{}
    strStack := []string{}
    cur := ""
    num := 0
    for i := 0; i < len(s); i++ {
        c := s[i]
        switch {
        case c >= '0' && c <= '9':
            num = num*10 + int(c-'0')
        case c == '[':
            countStack = append(countStack, num)
            strStack = append(strStack, cur)
            num, cur = 0, ""
        case c == ']':
            k := countStack[len(countStack)-1]
            countStack = countStack[:len(countStack)-1]
            prev := strStack[len(strStack)-1]
            strStack = strStack[:len(strStack)-1]
            cur = prev + strings.Repeat(cur, k)
        default:
            cur += string(c)
        }
    }
    return cur
}
```

```python
class Solution:
    def decodeString(self, s: str) -> str:
        count_stack, str_stack = [], []
        cur, num = "", 0
        for c in s:
            if c.isdigit():
                num = num * 10 + int(c)
            elif c == '[':
                count_stack.append(num)
                str_stack.append(cur)
                num, cur = 0, ""
            elif c == ']':
                k = count_stack.pop()
                cur = str_stack.pop() + cur * k
            else:
                cur += c
        return cur
```
