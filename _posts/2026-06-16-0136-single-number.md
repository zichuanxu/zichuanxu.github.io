## 题目描述

给定非空数组，除一个元素只出现一次外，其余每个元素均出现两次。找出那个只出现一次的元素。要求线性时间、常数空间。

**示例：** `nums = [4,1,2,1,2]` → `4`。

## 思路分析

**核心思想：** 异或。`a ^ a = 0`、`a ^ 0 = a` 且满足交换律，全部元素异或后成对的相互抵消，只剩那个唯一元素。

**详细步骤：**

1. `res = 0`。
2. 把所有元素依次异或进 `res`，结果即答案。

```go
func singleNumber(nums []int) int {
    res := 0
    for _, n := range nums {
        res ^= n
    }
    return res
}
```

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        res = 0
        for n in nums:
            res ^= n
        return res
```
