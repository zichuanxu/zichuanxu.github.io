## 题目描述

给定含 `0`、`1`、`2` 的数组（代表红白蓝），原地排序使相同颜色相邻、按红白蓝排列。要求一趟扫描、常数空间。

**示例：** `nums = [2,0,2,1,1,0]` → `[0,0,1,1,2,2]`。

## 思路分析

**核心思想：** 荷兰国旗三指针。`lo` 指向下一个 0 应放处、`hi` 指向下一个 2 应放处，`mid` 扫描：遇 0 与 `lo` 换、遇 2 与 `hi` 换、遇 1 前移。

**详细步骤：**

1. `lo = mid = 0, hi = n-1`。
2. `nums[mid] == 0`：与 `lo` 交换，`lo++, mid++`。
3. `== 1`：`mid++`；`== 2`：与 `hi` 交换，`hi--`（不动 `mid`，因换来的值还需判断）。

```go
func sortColors(nums []int) {
    lo, mid, hi := 0, 0, len(nums)-1
    for mid <= hi {
        switch nums[mid] {
        case 0:
            nums[lo], nums[mid] = nums[mid], nums[lo]
            lo++
            mid++
        case 1:
            mid++
        case 2:
            nums[mid], nums[hi] = nums[hi], nums[mid]
            hi--
        }
    }
}
```

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        lo, mid, hi = 0, 0, len(nums) - 1
        while mid <= hi:
            if nums[mid] == 0:
                nums[lo], nums[mid] = nums[mid], nums[lo]
                lo += 1
                mid += 1
            elif nums[mid] == 1:
                mid += 1
            else:
                nums[mid], nums[hi] = nums[hi], nums[mid]
                hi -= 1
```
